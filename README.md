This program is a password cracking game in python. The game has difficulty levels from 1 to 3. There are 3 ways to crack passwords, but if you fail, the game is over. In the future, my goal will be to try to make it as realistic as possible.


Ez a program egy jelszófeltörő játék Pythonban. A játék nehézségi szintje 1-től 3-ig terjed. Háromféleképpen lehet feltörni a jelszavakat, de ha nem sikerül, a játéknak vége. A jövőben az lesz a célom, hogy a lehető legrealisztikusabb legyen.
