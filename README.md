This is a CTF simulation game where we can get acquainted with the hacker world and how it works inside and out. After each successful hack in the game, we get XP, which we can exchange for stronger and faster attack commands. Both English and Hungarian languages ​​are available in the game settings, and we can also earn achievements during the game. In the settings, we can set the difficulty level and colors as desired. There will be further developments in the future.


Ez egy CTF szimulácíós játék ahol megismerkedhezűnk a hacker világgal hogy körül belűl hogyan működik. A játékban mindenegyes sikeres hackelés után XP-t kapunk amiket be tudunk váltani erősebb és gyorsabb támadási parancsokra. A játék beállításában elérhető az angol és a magyar nyelv is, továbbá achivmenteket is szerezhetűnk a játék folyamán. A beállításoknál tudjuk a nehézségi szintet és a színeket állítani tetszés szerint. További fejlesztések is lesznek hozzá a későbbiekben.
